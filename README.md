# Insurance & Life Expectancy Estimation

## Overview

This project applies machine learning techniques to model and predict life expectancy / insurance-related outcomes using structured tabular data.

The objective is to demonstrate regression modeling, feature analysis, and evaluation techniques in a real-world predictive setting.

## Problem Statement

Insurance pricing and life expectancy estimation rely on statistical modeling of demographic and risk-related variables.

This project explores:

- Data preprocessing and cleaning  
- Feature engineering and transformation  
- Training regression models  
- Evaluating predictive performance
